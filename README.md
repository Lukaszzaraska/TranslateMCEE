# TranslateMCEE

- T1 --> poziom nowicjusz
- T2 --> poziom zaawansowany
- T3 --> poziom ekspert

1. Stwórz plik z rozszerzeniem .md 
2. Umieść nazwe pliku w pxt.json
3. Zmodyfikuj link w plikach gry (blocks.mcfunction)
